S'il vous plait afin de pouvoir correctement utiliser l'application merci de suivre ce mini-tutoriel :

Dans un terminal Bash : 

	Lancer MariaDB sur le serveur de votre choix (par défaut à l'université il s'agit de DB+votre nom
	ouvrez le dossier creation_DATABASE et ouvre les deux fichiers dedans.
	copier et coller le contenue de creationLivreExpress.sql (ATTENTION,si vous ne voulez pas créer une nouvelle base de donnée, il faut que vous
	utilisiez une base de données de votre choix, donc ne copier pas les 3 premières lignes !) dans le terminal Bash sous mariaDB.
	Faite de même avec jeucomplet.sql
	ouvrez un nouveau terminal bash dans le dossier SAE_JAVA/
	tapper la commande ./exec.sh
